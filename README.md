# 📡 Port Status Checker

## 📌 Task ID
task2

## 📖 Description
A simple Python tool that checks whether a specific port on a given host is OPEN or CLOSED using socket programming.

---

## ⚙️ Features
- Check open/closed ports
- Supports IP and domain names
- Input validation (1–65535)
- Error handling
- Fast response using timeout

---

## 🛠️ Tech Used
- Python 3
- socket module
